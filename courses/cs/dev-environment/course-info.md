Before you write a single line of code, you need the right tools. This course walks you through everything a professional developer has on their machine — and why each tool matters.

## What You'll Set Up

- Visual Studio Code with the essential extensions
- The terminal and basic command-line navigation
- Node.js and npm
- Cursor IDE for AI-assisted development

## Who This Is For

Complete beginners who have never written code before, or developers switching to a new machine and wanting a clean, modern setup.

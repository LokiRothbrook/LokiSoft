Version control is how developers save their work, collaborate without overwriting each other, and roll back mistakes. Git is the industry standard — and GitHub is where the world's code lives.

## What You'll Learn

- What version control is and why it matters
- Installing and configuring Git
- Core Git workflow: init, add, commit, log
- Branching and merging strategies
- Working with remote repositories on GitHub
- Pull requests, issues, and .gitignore

## Prerequisites

Complete the **Developer Environment Setup** course first — you'll need a working terminal and Node.js installed.

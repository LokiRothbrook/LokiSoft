Every webpage on the internet is built on HTML. Before you can style, script, or deploy anything, you need to understand how browsers read and display markup.

## What You'll Learn

- How HTML documents are structured
- Headings, paragraphs, and text formatting
- Links, images, and media embedding
- Lists, tables, and forms
- Semantic HTML and why it matters for accessibility and SEO
- HTML entities, attributes, and best practices

## Projects

You'll build a static profile page and a full freelancer landing page by hand — no frameworks, no shortcuts.

## Prerequisites

Complete the **Developer Environment Setup** course first so you have a code editor and can preview your work in a browser.

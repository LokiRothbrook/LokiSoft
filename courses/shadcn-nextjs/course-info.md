This course takes you from absolute beginner to full-stack developer. No prior coding experience required — just a computer, a browser, and the willingness to learn.

You will build a foundation in HTML, CSS, and JavaScript before climbing into TypeScript, React, and Next.js. By the end, you will have built and deployed a real full-stack application with a database, authentication, and a polished shadcn/ui interface.

## What You'll Build

- Static marketing pages with pixel-perfect layouts
- Interactive React components with state, forms, and animations
- A full-stack Task Management Dashboard with user authentication, database storage, and real-time updates

## Technologies Covered

- HTML5, CSS3, JavaScript (ES2024)
- TypeScript
- React 19 & Next.js 15 (App Router)
- shadcn/ui & Tailwind CSS v4
- Prisma ORM & PostgreSQL
- Vercel deployment pipeline

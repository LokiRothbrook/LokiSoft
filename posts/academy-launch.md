# LokiSoft Academy is Open

We have been talking about this for a while. Today it's real — **LokiSoft Academy** is officially open and free for everyone.

[Go to LokiSoft Academy →](/academy)

> *"My people are destroyed for lack of knowledge."*
> — Hosea 4:6 (KJV)

We believe knowledge belongs to everyone. Not just people who can afford a $500 course, or who live near a university, or who already have a foot in the door. Programming is one of the most empowering skills a person can learn, and there is no good reason it should be locked behind a paywall. The Academy is our answer to that.

---

## What Is the Academy?

LokiSoft Academy is a structured, self-paced learning platform built directly into this website. No separate account. No subscription. No ads. Just good content organized into courses and lessons that build on each other.

Courses are divided into tracks. Right now we're launching with the **Computer Science** track, which is designed to take someone with zero experience and give them a genuine foundation in modern web development.

---

## What's Available Now

The Computer Science track currently has three full courses:

### How the Web & Your Dev Environment Works
Start here if you've never written a line of code. This course covers:
- How the web actually works — browsers, servers, HTTP, DNS
- Installing and configuring VS Code
- The essential extensions that make development productive

### Git & Version Control
The tool that every developer uses every day:
- What version control is and why it matters
- Setting up Git and your first repository
- Committing, branching, and working with GitHub
- A practical workflow for real projects

### HTML Fundamentals
The language of the web:
- Document structure and the building blocks of every webpage
- Text, links, images, and media
- Lists, tables, and organizing content
- Semantic HTML for accessibility and SEO

Each course is broken into focused lessons that you can complete at your own pace. No timers, no deadlines, no pressure.

## What's Coming

This is just the start. We're actively building more content:

- **CSS Fundamentals** — Styling, layout, Flexbox, Grid, responsive design
- **JavaScript** — The programming language of the web, from basics to async
- **React** — Building modern, component-driven user interfaces
- **Next.js** — Full-stack web applications the modern way
- **Databases & APIs** — How the backend works and how to build it

We'll keep adding courses as we build them. The Computer Science track will grow into a complete path from zero to full-stack developer.

---

## How to Get Started

Head to [LokiSoft Academy](/academy) and pick a course. That's it. No sign-up, no credit card, no email required.

If you're brand new, start with the Dev Environment course — it sets up everything you need to follow along with the rest of the content hands-on. If you have some experience, jump into whichever course covers the gap in your knowledge.

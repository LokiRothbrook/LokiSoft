## What Is LokiASAM?

LokiASAM is a **free, open-source desktop application** for managing dedicated ARK: Survival Ascended servers on **Windows and Linux**. It's built with Tauri v2, Next.js, and Rust — which means it runs natively, uses minimal resources, and feels fast.

No browser required. No monthly subscription. No cloud dependency. Your servers, your machine, your control.

---

## What It Can Do Right Now

Here's what's available in the current beta:

- **Server management** — Create, start, stop, restart, clone, and delete servers from one dashboard
- **SteamCMD integration** — Automated install, update, and validation with a shared cache that speeds up multi-server setups
- **Server configuration** — Edit `GameUserSettings.ini` and `Game.ini` with a structured editor or raw text area
- **RCON console** — Live terminal with command history, saved presets, and broadcast support across all servers
- **Live log viewer** — Tail server output in real time with filtering and auto-scroll
- **Mod management** — Install mods by ID, drag to reorder, toggle per server, and browse CurseForge from inside the app
- **Scheduled automation** — Cron-based schedules for backups, updates, restarts, and RCON broadcasts
- **Backup and restore** — ZIP backups with retention settings so you never lose a save
- **Cluster support** — Group servers with a shared directory for cross-server progression
- **Discord & email notifications** — Webhooks and SMTP alerts for server events
- **Proton-GE support** — Automatic detection and download on Linux for running the Windows server binary
- **System tray** — Minimizes to tray while your servers keep running in the background
- **Auto-updates** — The app keeps itself current automatically
